# Rule Parity Artifacts

Use this folder to prove no-loss migration when splitting or reorganizing plugins.

## Files

- `skills-inventory.md`: inventory of legacy rules from skill files
- `conduct-inventory.md`: inventory of legacy rules from conduct docs
- `mapping.md`: one-to-one mapping from legacy rules to new owner plugins

## Gate

Do not deprecate old plugin structures until all mapped rules are preserved (or explicitly waived with rationale).
